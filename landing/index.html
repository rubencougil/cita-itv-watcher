Node.js ≥ 18 · Playwright · Telegram · Windows · macOS · Linux

⏰ ITV Watcher

Avisa por Telegram cuando Applus+ libere una cita de ITV.

Proyecto público en GitHub que monitoriza la disponibilidad de citas ITV en Applus+, respeta tu fecha mínima y te avisa por Telegram en cuanto detecta huecos compatibles, con captura incluida.

El problema

Conseguir cita para la ITV suele implicar entrar en la web una y otra vez para comprobar si han aparecido huecos. Cuando no hay un aviso automático, la vigilancia manual se vuelve repetitiva y fácil de olvidar.

ITV Watcher hace esa vigilancia por ti y te notifica por Telegram en el momento exacto en que detecta una cita compatible con tu filtro.

Cómo funciona

  1. Abre la web de Applus+ y completa el arranque

    Rellena la matrícula y, si la web lo pide, introduce los datos de contacto una sola vez.

  2. Vigila el calendario cada N minutos

    Detecta cambios, extrae las fechas visibles y aplica la fecha mínima configurada.

  3. Te avisa por Telegram con captura

    Envía un mensaje con la información relevante y una captura cuando aparece una cita compatible.

Inicio rápido

$ git clone https://github.com/rubencougil/cita-itv-watcher.git
$ cd cita-itv-watcher
$ npm install && npx playwright install chromium
$ cp .env.example .env  # añade matrícula, Telegram y fecha mínima
$ npm start

Para una sola comprobación: RUN_ONCE=true npm start.
