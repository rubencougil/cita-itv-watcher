⏰ ITV Watcher

Monitoriza la ITV y recibe aviso por Telegram cuando aparezca un hueco.

Software público que monitoriza la disponibilidad de citas para la ITV en Applus+ y te avisa por Telegram en cuanto se libera un hueco, sin que tengas que estar pendiente de la página.

El problema

Conseguir cita para la ITV suele implicar esperar y revisar la web una y otra vez. No existe un sistema oficial que te avise cuando aparecen huecos disponibles, así que la vigilancia manual se vuelve la única opción.

ITV Watcher monitoriza la página por ti y te notifica por Telegram en el momento exacto en que detecta una cita compatible con tu filtro.

Cómo funciona

1

Abre un navegador con Playwright

Rellena la matrícula y completa el flujo inicial una vez si el sitio lo solicita.

2

Monitoriza la página cada N minutos

Detecta cambios, extrae las fechas disponibles y aplica tu fecha mínima configurada.

3

Te avisa por Telegram

Envía un mensaje cuando aparece una cita dentro de tu rango aceptable y mantiene un heartbeat.

Inicio rápido

1

Clona el repositorio

git clone https://github.com/rubencougil/cita-itv-watcher.git
cd cita-itv-watcher
2

Instala Playwright y configura el entorno

npm install
npx playwright install chromium
cp .env.example .env

Añade tu matrícula, datos de contacto y credenciales de Telegram.

3

Arranca el watcher

npm start

Si quieres una sola comprobación: RUN_ONCE=true npm start
