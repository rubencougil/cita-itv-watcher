⏰ ITV
Watcher

Software que monitoriza la disponibilidad de citas para la ITV en Applus+ y te avisa por Telegram en cuanto se libera un hueco, sin que tengas que estar pendiente de la página.

⭐ Ver en GitHub 📦 Descargar ZIP

El problema

Renovar la ITV suele implicar revisar la web una y otra vez para encontrar huecos, y no existe un sistema oficial que te avise cuando aparecen citas disponibles.

ITV Watcher monitoriza la página por ti y te notifica por Telegram en el momento exacto en que se libera una cita, sin que tengas que estar pendiente. Si quieres abrirla manualmente, puedes ir a la página de citas de Applus+.

Cómo funciona

1

Abre un navegador con Playwright

Rellenas la matrícula y, si hace falta, los datos de contacto una vez.

2

Monitoriza la página cada N minutos

Detecta cambios y extrae las fechas disponibles automáticamente.

3

Te avisa por Telegram

Mensaje + captura de pantalla cuando aparece una cita dentro de tu rango aceptable.

Inicio rápido

bash 
git clone https://github.com/rubencougil/cita-itv-watcher.git
cd cita-itv-watcher
npm install && npx playwright install chromium
cp .env.example .env  # añade tu matrícula, Telegram y fecha mínima
npm start

Para una sola comprobación: RUN_ONCE=true npm start.
